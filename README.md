# Algorithms and Data Structures

A collection of university laboratory assignments covering essential abstract data types, sorting mechanisms, greedy approaches, and fundamental graph algorithms.

## Laboratory Works Breakdown
* **Lab 1:** Stack data structure and fundamental stack operations.
* **Lab 2:** Queue data structure and fundamental queue operations.
* **Lab 3:** Hash tables implementation and collision resolution.
* **Lab 4:** Binary search trees (BST) and tree traversal operations.
* **Lab 5:** Insertion Sort algorithm.
* **Lab 6:** Merge Sort algorithm.
* **Lab 7:** Heap Sort algorithm.
* **Lab 8:** Quick Sort algorithm.
* **Lab 9:** Greedy algorithms and formal proofs of correctness.
* **Lab 10:** Huffman coding algorithm for data compression.
* **Lab 11-12:** Minimum Spanning Tree (MST) generation via Kruskal's and Prim's algorithms.
* **Lab 13:** Graph traversal and shortest path problems.
* **Lab 14:** Combinatorial algorithms.
* **Lab 15:** Linear and non-linear programming optimization problems.
* **Lab 16:** Overview of core un-computable and undecidable problems.
